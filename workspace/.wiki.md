# Project Summary
AMIGOS DE TRON is a decentralized crowdfunding platform built on the TRON blockchain, empowering entrepreneurs to secure funding through a smart contract-driven global matrix system. The platform enhances sustainability and user engagement via a TRX token burning mechanism and an automatic renewal system for commission plans, ensuring continuous support for users.

# Project Module Description
The project consists of several functional modules:
- **Landing Page**: Overview of the platform's objectives and features.
- **User Dashboard**: Displays user statistics, referral links, matrix visualizations, and wallet connection features.
- **Admin Panel**: Restricted access for the owner, providing insights into project metrics and user activity.
- **Multilingual Support**: Available in Spanish, English, and Portuguese.
- **Wallet Connection**: Enables users to connect their TronLink wallets for account activation.
- **Referral System**: Generates unique referral links for users, optimized for scalability.
- **Transaction History**: Lists all user transactions.
- **Matrix System**: Implements a single 1x2 global matrix structure for user engagement and earnings.
- **Commission Plans**: Features three distinct commission plans with automatic renewal.
- **Reinvestment Fund**: Automatically creates new accounts based on the fund's balance.
- **TRX Burning Mechanism**: Facilitates both direct and indirect burning of TRX through transaction fees.
